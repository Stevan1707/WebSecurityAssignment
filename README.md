# 密码学实验项目：RSA与Diffie-Hellman算法的实现与分析

## 项目概述

本项目实现了RSA加密算法的攻击与防御方法，以及Diffie-Hellman密钥交换协议的实现与安全性分析。

## 已完成内容

### RSA算法部分

- **攻击方法**：Wiener攻击（针对小私钥d的RSA攻击）
- **防御方法**：
  - 抬高私钥d的下界
  - 拉大素数p和q的差距
  - 保证公钥e足够大

### D-H算法部分

- **攻击方法**：中间人攻击
- **防御方法**：认证防御
